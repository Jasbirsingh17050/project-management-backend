# Project Management Backend System

A basic Project Management Backend System built using Django and Django REST Framework.

## Features

- User Management CRUD APIs
- Project Management CRUD APIs
- Task Management CRUD APIs
- Proper ForeignKey relationships
- JSON responses
- Input validation
- SQLite database

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
